# Pokemon GO servers status

Check the status of the Pokemon GO servers from your Appindicator (Linux)

![Demo](demo.png)

color code:

- green: server up
- orange: server unstable
- red: server down

Install `python-gi` with `sudo apt-get install python-gi`
